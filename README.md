## Modelagem Conceitual – E-commerce

Este projeto apresenta um esquema conceitual de banco de dados para um cenário simplificado de E-commerce, modelado a partir de um contexto de minimundo. O foco foi representar as principais entidades e seus relacionamentos, refletindo as regras básicas do negócio.

A modelagem foi construída com base no Modelo Entidade-Relacionamento (MER). Apesar de se tratar de um modelo conceitual, foram indicadas PKs e FKs para facilitar a compreensão das relações entre as entidades.

O objetivo é servir como base para evoluções futuras do sistema e reforçar a importância da modelagem no processo de desenvolvimento de bancos de dados.
